Projects

Here are some of the key projects I've worked on as a Researcher at the COSMOS Lab, Columbia University.

Distributed Vision Language Models

This paper introduces a distributed architecture for VLMs, addressing the challenges of deploying these models on edge devices. By partitioning the model architecure between heterogenous devices, the approach improves throughput by up to 33% over cloud-only solutions and enhances computational efficiency without requiring model compression.

Here are some of the other projects I have worked on:

Arrhythmia Detection using Vision Transformers

This study investigates using vision transformers with knowledge distillation to detect cardiac arrhythmias in 12-lead ECG images, enabling early detection in non-clinical settings.

Real-time Pose Estimation and Correction

This project develops a real-time 3D pose estimation and posture correction app for yoga on edge devices using MediaPipe, deployed on NVIDIA® Jetson Nano™. It provides real-time feedback and guidance to ensure proper alignment, demonstrating the potential of edge computing in health and wellness.

Brain Tumor Segmentation using U-Net Architecture

This project uses the U-Net architecture for brain tumor segmentation on MRI images from the BraTS 2020 dataset, classifying pixels into categories like Normal, Tumor Core, Edema, and Enhancing Tumor.

Two Lies, One Truth: Can you outsmart the model?

Try it now - Launch!
An interactive web app was developed using Streamlit to showcase OpenAI's CLIP model for image understanding. Users upload an image and provide three descriptions, one of which is true. The app uses CLIP to identify the most accurate description with the highest probability.

Asteria Aerospace
During my senior year of undergrad, I worked at Asteria Aerospace as an Embedded Software Intern from June to December 2020. I developed a solution for autonomous indoor navigation in the absence of GPS signals for Quadcopter drones, with visual-inertial odometry and SLAM (Simultaneous Localization and Mapping) algorithms. Using an Intel T265 camera and NVIDIA Jetson Xavier NX edge devices, I engineered a system to process and transmit spatial data from the camera to the simulation software utilizing ROS as an abstraction layer between the camera hardware and the simulation software.
