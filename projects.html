Arrhythmia Detection using Vision Transformers

This study explores the use of vision transformers for detecting cardiac arrhythmias in 12-lead ECG images, aiming to facilitate early detection and treatment from non-clinical settings. It investigates transformer architectures like BEiT, Swin Tiny, Google ViT, and DEiT for their robust feature extraction capabilities in medical image analysis. The approach includes a knowledge distillation framework where a larger pre-trained transformer (teacher) transfers learned representations to a compact CNN (student) model, optimizing performance and computational efficiency.
Keywords: Vision Transformers, CNNs, Knowledge Distillation

Real-time Pose Estimation and Correction

This project involves developing a real-time 3D pose estimation and posture correction application for yoga on edge devices, using MediaPipe for optimal accuracy and efficiency. Deployed on NVIDIA® Jetson Nano™, it provides real-time feedback and corrective guidance, integrating a user-friendly interface for seamless interaction. The application accurately estimates body positions and offers corrective suggestions to ensure proper alignment and form. This showcases the potential of edge computing in enhancing health and wellness through interactive and immersive experiences.
Keywords: Pose Estimation, Deep Learning on Edge, Mediapipe, Nvidia Jetson Nano

Brain Tumor Segmentation using U-Net Architecture

This project involves utilizing the U-Net architecture to perform brain tumor segmentation on MRI images from the BraTS 2020 dataset. The goal was to classify each pixel into categories such as Normal, Necrotic and Non-Enhancing Tumor Core, Peritumoral Edema, and GD-enhancing tumor. The project involved two models - a pre-trained encoder model with MobileNetV2 as the encoder and a trained from scratch model (TFS). The TFS model showed superior performance over the pre-trained encoder model.
Keywords: Brain Tumor Segmentation, U-Net Architecture, MobileNetV2

Two Lies, One Truth: Can you outsmart the model?

An interactive web application was developed using Streamlit to demonstrate the capabilities of OpenAI's CLIP model in image understanding. The app allows users to upload an image and input three descriptions, with one being true. Utilizing the CLIP model, the app processes the image and descriptions, identifying the most accurate description with the highest probability. This project showcases the ability to integrate advanced machine learning models into user-friendly applications, emphasizing practical applications of vision-language models in real-world scenarios.
